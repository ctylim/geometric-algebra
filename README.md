# geometric-algebra
Code Scraps for Conformal Geometric Algebra

### Overview

This repository is a playground for learning conformal geometric algebra with [clifford](https://github.com/pygae/clifford).

### Installation

- python 3.6

```bash
$ pip install -r requirements.txt
```

### Run Notebook

```bash
$ jupyter notebook
```
